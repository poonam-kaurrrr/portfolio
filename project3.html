Axis Asian Bank

The Axis Asian Bank System UI follows an Earthy Elegance theme that blends professionalism with a warm, approachable feel. It uses a grounded color palette featuring dark grayish for the main background, soft yellows and olive greens for the logo accents, muted lime green and light beige tones to highlight key brand elements, and white for clear, readable text. Thoughtful typography and balanced contrast guide users smoothly through banking tasks. This project was developed using tools such as Canva.
